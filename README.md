Programmatically Emails to a list of recipients using a google 
email account.

To setup you need to setup App access to your google account using:
Make sure 2-step verification is setup and on,
Add a app password with:
Manage Account -> Security -> 2-step Verification -> App passwords.

To setup program access to credentials, email subject and body, 
and list of recipients, make and add information to data.json 
file similar to the data-example.json file.

Run mailer.js to run.
